# Cake.Coveralls

[![License](http://img.shields.io/:license-mit-blue.svg)](http://cake-contrib.mit-license.org)

Cake.Coveralls is an Addin for [Cake](http://cakebuild.net/) with ability to push Code Coverage results to Coverall.io.
